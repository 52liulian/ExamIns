# 考试指令播放系统

基于Python和PyQt5实现的考试指令播放系统。

## 功能特点

1. **两种考试模式**
   - 高考模式：严格按照高考时间安排
   - 普通模式：可自定义考试时间

2. **自动播放指令**
   - 根据考试时间自动播放相应的语音指令
   - 支持考前和考后指令
   - 实时显示系统时间和下一条指令

3. **完整的指令系统**
   - 包含完整的考试流程指令（入场检查、身份验证、分发试卷、开始答题、考试提醒、考试结束等）
   - 指令内容基于四川省高考指令系统

4. **图形化界面**
   - 简洁直观的用户界面
   - 支持查看完整指令列表
   - 实时显示当前状态

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. **启动程序**
   ```bash
   python main.py
   ```

2. **选择考试模式**
   - 高考模式：选择科目后自动使用标准高考时间
   - 普通模式：选择科目并自定义开始和结束时间

3. **考试科目**
   - 高考模式：语文、数学、文科综合、理科综合、外语
   - 普通模式：语文、数学、思想政治、外语、物理/历史、信息/通用技术、化学/地理、地理、生物、物理、历史、信息技术、通用技术、其他

4. **考试时间**
   - 高考模式：
     - 上午科目：09:00 - 11:30
     - 下午科目：15:00 - 17:00
   - 普通模式：可根据科目选择预设时间或自定义时间

5. **指令播放**
   - 程序会根据考试时间自动计算并播放相应的语音指令
   - 可点击"查看指令列表"查看所有指令的播放时间
   - 可点击"停止"按钮停止当前考试并返回选择界面

## 注意事项

1. 确保Mp3文件夹中包含所有必要的音频文件
2. 音频文件命名必须与程序中定义的一致（p1.mp3 - p10.mp3）
3. 程序使用系统时间，请确保系统时间准确
4. 普通模式下，考试时间间隔不能少于1小时

## 技术栈

- Python 3.x
- PyQt5：图形界面框架
- pygame：音频播放
